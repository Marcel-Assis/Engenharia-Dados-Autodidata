# Engenharia de Dados Autodidata

Este repositório registra minha jornada de estudos em **Engenharia de Dados** de forma **autodidata**.

Ao longo do tempo, vou estudando desde fundamentos de programação e manipulação de dados até tópicos mais avançados, como banco de dados, computação em nuvem, APIs, testes, machine learning, deploy de aplicações e projetos finais.

## Objetivo

- Aprender engenharia de dados de forma estruturada e contínua  
- Praticar por meio de exercícios, experimentos e pequenos projetos  
- Registrar toda a evolução através de **commits** que documentam cada etapa do aprendizado  

Sinta-se à vontade para acompanhar, abrir issues com sugestões ou compartilhar ideias de melhoria.
